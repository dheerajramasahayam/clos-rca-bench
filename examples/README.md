# Example Outputs

This directory contains lightweight, shareable artifacts that make the repository easier to inspect without rebuilding the full benchmark first.

## Included examples

- `closrca_bench_sample_windows.csv`: sample benchmark window manifest rows
- `closrca_bench_snapshot.json`: compact dataset and leaderboard snapshot
- `closrca_bench_leaderboard.png`: rendered leaderboard image for slides and applications

## Full outputs elsewhere in the repo

- Leaderboard CSV: `results/closrca_bench_leaderboard.csv`
- Target confusion matrix: `graphs/topology_target_cm.png`
- Model comparison figure: `graphs/topology_model_comparison.png`
- Notebook demo: `notebooks/closrca_bench_demo.ipynb`
